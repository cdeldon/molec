Calculate short-range interaction force using the Lenard-Jones potential.

This function computes the force between all the particiles using a straightforward N^2 approach. In addition the potential energy is computed.

See also
http://polymer.bu.edu/Wasser/robert/work/node8.html for derivation of the forces.
